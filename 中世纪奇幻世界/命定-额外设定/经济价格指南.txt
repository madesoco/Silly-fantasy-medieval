<经济价格指南>
货币体系:
  单位: G (非金/银/铜币，仅通用价值计量单位)
  核心概念:
    定义: G并非特定实体货币，而是源自跨势力商会为便利贸易约定的价值锚点，现已成为全大陆通用抽象计价单位
    现状: 各国物理货币材质面额各异，交易时皆按标准G值结算
    精度: 最小单位1G，不可拆分，向下取整

交易原则:
  定价独立性:
    价值锚定: 物品价格仅由价格矩阵参考决定，严禁根据买方当前持有资产量动态调整
    购买力恒定: 1G购买力客观固定，不因持有者财富多寡产生通胀
  消费行为逻辑:
    理性默认: 无论资产多寡，默认遵循理性商业逻辑(核对/找零/议价)
    挥霍禁令: 严禁主动放弃找零或支付不合理溢价
    认知修正: 巨额财富仅提升支付能力上限，不改变对基础物品低廉价格的客观认知
  高阶流通:
    货币局限: 史诗以上物品通常无法通过单纯货币购买，多为以物易物或顶级拍卖流转，此处价格仅体现价值

阶级现金流:
  说明: 前值为随身携带日常支付上限，后值为资产调动极限(需时间变卖或调取国库)
  大国统治者: 5万-20万 / 10亿-100亿+ (足以发动大陆级战争或建设奇观)
  小国统治者: 1万-5万 / 1亿-8亿 (可负担顶级城防与传说级战略物资)
  大师工匠/大贵族: 2000-1万 / 1000万-1亿 (足以通过变卖领地或稀世作品购置史诗装备)
  富商/低级贵族: 1000-3000 / 100万-800万 (足以购置城市房产与稀有级魔法道具)
  普通工匠/商人/中产: 50-300 / 5万-40万 (足以负担子女基础魔法教育或乡镇房产)
  平民: 5-30 / 1000-5000 (仅能维持温饱与极少量储蓄)
  奴隶/底层: 0-5 / 0 (自身即为财产)

定价排序逻辑:
  说明: 同品质级别下决定价格在区间内的具体落点，从下限往上限浮动
  物品价格权重:
    数值面板: 基础白板 < 词条优良 < 效果极品 < 特殊效果
    地理因素: 原产地 < 偏远村镇 < 普通城市 < 繁华都市/要塞
    制作工艺: 量产制式 < 匠人手作 < 大师定制 < 附魔/刻印
    物品类型: 消耗品 < 轻型 < 重型 < 特殊
  任务报酬权重:
    风险程度: 采集/跑腿 < 驱逐/守卫 < 讨伐/猎杀 < 探索未知/决战
    雇主财力: 平民/村长 < 商会/行会 < 贵族/领主 < 王室/跨国组织
